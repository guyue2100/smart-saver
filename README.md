# 彦仔宝库 (SmartSaver) 💎

这是一个专为儿童设计的游戏化零花钱管理 App，旨在培养财商和储蓄习惯。应用基于 React 构建，支持 PWA，可以安装在手机上使用。

## ✨ 核心功能

- **📊 资产总览**：清晰展示总资产、零钱和愿望储蓄。
- **📈 趣味复利**：支持阶梯利率或固定利率，每周日自动结算利息，让孩子理解“钱生钱”。
- **🎯 愿望清单**：设立目标（如乐高），存钱进度可视化。
- **🔥 自律挑战**：连续几周不乱花钱可获得额外奖励。
- **🏅 勋章系统**：达成成就解锁炫酷勋章。
- **📱 手机安装**：支持 iOS 和 Android 安装到主屏幕（PWA）。

## 🚀 如何部署 (免费)

为了在手机上使用，你需要将代码部署到支持 HTTPS 的平台。推荐使用 **Vercel**：

1. 将本项目所有文件上传到您的 GitHub 仓库。
2. 登录 [Vercel](https://vercel.com/)，选择 "Add New Project"。
3. 导入您的 GitHub 仓库。
4. Vercel 会自动识别 Vite 配置，直接点击 "Deploy" 即可。
5. 部署完成后，使用手机访问生成的链接即可安装 App。

## 🛠️ 本地开发

如果您想在电脑上修改代码：

1. 安装依赖：`npm install`
2. 启动开发服务器：`npm run dev`
3. 构建打包：`npm run build`
